## 1. 로그인

관리자 페이지에 접속하려면 먼저 로그인이 필요합니다.

### 접속 방법

1. 브라우저에서 **사이트주소/admin/login** 으로 이동합니다.
   - 예: `https://dongnam.church/admin/login`
2. 이메일과 비밀번호를 입력합니다.
   - 기본 계정: `admin@dongnam.church` / `admin1234`
3. **로그인** 버튼을 클릭합니다.
4. 로그인에 성공하면 자동으로 대시보드 페이지로 이동합니다.

### 로그아웃

- 관리자 페이지 왼쪽 사이드바 하단의 **로그아웃** 버튼을 클릭합니다.

### 참고 사항

- 이메일 또는 비밀번호가 올바르지 않으면 빨간색 오류 메시지가 표시됩니다.
- 로그인하지 않은 상태에서 관리자 페이지에 접근하면 자동으로 로그인 페이지로 이동합니다.

## 2. 대시보드

로그인 후 가장 먼저 보이는 페이지입니다.

### 대시보드에서 확인할 수 있는 정보

- **설교 영상** -- 등록된 설교 총 개수
- **설교 원고** -- 작성된 칼럼 총 개수
- **찬양 콘티 & 가사 PPT** -- 업로드된 찬양 콘티 및 가사 PPT 총 개수
- **주보/PPT** -- 업로드된 주보 및 PPT 총 개수
- **갤러리** -- 업로드된 갤러리 사진 총 개수
- **최근 7일 방문자 차트** -- 일별 방문수 막대 그래프
- **인기 페이지 순위** -- 최근 7일 기준 가장 많이 방문한 페이지 목록

### 빠른 작업

대시보드 하단에 자주 사용하는 작업 바로가기가 있습니다:

- **+ 설교 등록** -- 새 설교를 등록합니다.
- **+ 콘티 업로드** -- 찬양 콘티 및 가사 PPT 파일을 업로드합니다.
- **+ 주보 업로드** -- 주보/PPT 파일을 업로드합니다.
- **+ 설교 원고 작성** -- 새 설교 원고(칼럼)를 작성합니다.

### 사이드바 메뉴

화면 왼쪽의 사이드바에서 모든 관리 메뉴에 접근할 수 있습니다:

| 메뉴 | 경로 | 설명 |
|------|------|------|
| 대시보드 | `/admin/dashboard` | 전체 현황 요약 + 방문자 차트 |
| 설교 관리 | `/admin/sermons` | 설교 등록/수정/삭제 |
| 찬양 콘티 & 가사 PPT | `/admin/praise` | 찬양 콘티 및 가사 PPT 업로드/관리 |
| 주보/PPT | `/admin/bulletins` | 주보 및 PPT 업로드/관리 |
| 설교 원고 | `/admin/columns` | 설교 원고(칼럼) 작성/관리 |
| 갤러리 | `/admin/gallery` | 교회 사진 업로드/관리 |
| 교회 일정 | `/admin/calendar` | 월간 일정 등록/삭제 |
| 기도 요청 | `/admin/prayer` | 기도 제목 확인 및 응답 처리 |
| 사이트 설정 | `/admin/settings` | 교회 정보 및 예배 시간 등 설정 |

사이드바 하단의 **사이트 보기** 링크를 클릭하면 실제 홈페이지(공개 페이지)를 확인할 수 있습니다.

## 3. 설교 관리

설교 영상 정보를 등록하고 관리하는 메뉴입니다.

### 설교 목록 확인

1. 사이드바에서 **설교 관리**를 클릭합니다.
2. 등록된 설교 목록이 표 형태로 표시됩니다.
   - **제목** -- 설교 제목
   - **성경** -- 성경 구절
   - **유형** -- 예배 유형 (주일예배, 수요예배 등)
   - **날짜** -- 설교 날짜
   - **관리** -- 수정/삭제 버튼

### 새 설교 등록

1. 설교 관리 페이지에서 오른쪽 상단의 **+ 새 설교** 버튼을 클릭합니다.
2. 아래 항목을 입력합니다:

| 항목 | 필수 여부 | 설명 |
|------|-----------|------|
| 설교 제목 | 필수 | 설교의 제목을 입력합니다. |
| 성경 구절 | 필수 | 본문 성경 구절을 입력합니다. (예: 요한복음 3:16) |
| 설교 날짜 | 필수 | 설교가 진행된 날짜를 선택합니다. |
| 예배 유형 | 필수 | 드롭다운에서 선택합니다: 주일예배, 주일학교, 수요예배, 금요예배, 특별예배 |
| 유튜브 URL | 선택 | 유튜브 설교 영상 URL을 붙여넣습니다. (예: `https://youtube.com/watch?v=...`) |
| 블로그 URL | 선택 | 관련 블로그 포스트 URL을 입력합니다. |
| 설교 요약 | 선택 | 설교 내용을 간략하게 요약합니다. |
| 시리즈 | 선택 | 시리즈명을 입력합니다. (예: 믿음 시리즈) |
| 태그 | 선택 | 쉼표로 구분하여 태그를 입력합니다. (예: 믿음,기도,은혜) |

3. **등록하기** 버튼을 클릭합니다.
4. 등록이 완료되면 자동으로 설교 목록 페이지로 이동합니다.

### 설교 수정

1. 설교 목록에서 수정하려는 설교의 **수정** 링크를 클릭합니다.
2. 수정할 내용을 변경합니다.
3. **수정하기** 버튼을 클릭합니다.

### 설교 삭제

1. 설교 목록에서 삭제하려는 설교의 **삭제** 버튼을 클릭합니다.
2. 확인 대화상자에서 **확인**을 클릭합니다.
3. 삭제된 설교는 복구할 수 없으므로 주의하세요.

### 유튜브 URL 안내

- `https://www.youtube.com/watch?v=VIDEO_ID` 형식의 URL을 입력하세요.
- `https://youtu.be/VIDEO_ID` 형식도 사용 가능합니다.
- URL을 입력하면 시스템이 자동으로 유튜브 영상 ID를 추출하여 홈페이지에서 영상 썸네일과 링크를 표시합니다.

## 4. 찬양 콘티 & 가사 PPT 관리

찬양 콘티 및 가사 PPT 파일을 업로드하고 관리하는 메뉴입니다. Google Drive에서 자동 동기화되며, 업로드된 파일은 성도들이 공개 페이지에서 다운로드할 수 있습니다.

### Google Drive 자동 동기화

찬양 콘티 파일은 Google Drive 폴더에서 자동으로 동기화됩니다.

- **자동 실행**: 매일 오전 6시 (Vercel Cron)
- **수동 실행**: 대시보드에서 **Google Drive 동기화** 버튼 클릭
- **지원 형식**: `.key` (Keynote) 파일 -- PDF로 자동 변환됨
- **중복 방지**: 이미 등록된 파일은 건너뜁니다.

### 파일명 규칙과 자동 태그

파일명에서 코드, 메세지, 절기 정보를 자동으로 추출합니다.

| 파일명 패턴 | 추출 결과 |
|------------|----------|
| `(A)찬양곡.key` | 코드: A |
| `(D:주제)찬양곡.key` | 코드: D, 메세지: 주제 |
| `(C,G:주제)찬양곡.key` | 코드: C,G, 메세지: 주제 |
| 파일명에 "성탄", "부활" 등 포함 | 절기 자동 감지 |

### 공개 페이지 필터 기능

성도들은 공개 찬양 페이지에서 다음 기능을 사용할 수 있습니다:

- **검색** -- 찬양곡 제목으로 검색
- **코드별 필터** -- 음악 코드(A, B, C, D, E, F, G 등)별 분류
- **메세지별 필터** -- 주제/메세지별 분류 (은혜, 성령, 부흥, 왕 등)
- **절기별 필터** -- 교회 절기별 분류 (성탄절, 부활절, 추수감사절 등)

### 콘티 목록 확인

1. 사이드바에서 **찬양 콘티 & 가사 PPT**를 클릭합니다.
2. 업로드된 콘티 목록이 표시됩니다.

### 새 콘티 수동 업로드

1. 찬양 콘티 & 가사 PPT 페이지에서 **+ 새 콘티** 버튼을 클릭하거나, 대시보드의 **+ 콘티 업로드** 바로가기를 사용합니다.
2. 아래 항목을 입력합니다:

| 항목 | 필수 여부 | 설명 |
|------|-----------|------|
| 제목 | 필수 | 콘티 제목을 입력합니다. (예: 2025년 4월 첫째주 찬양 콘티 & 가사 PPT) |
| 예배 날짜 | 필수 | 해당 예배 날짜를 선택합니다. |
| 파일 업로드 | 필수 | 콘티 파일을 선택합니다. |

3. **업로드** 버튼을 클릭합니다.

### 지원 파일 형식

- PDF (`.pdf`)
- 한글 (`.hwp`)
- Word (`.docx`)
- PowerPoint (`.pptx`, `.ppt`)
- 최대 파일 크기: **50MB**

### 콘티 삭제

1. 콘티 목록에서 삭제하려는 항목의 **삭제** 버튼을 클릭합니다.
2. 확인 대화상자에서 **확인**을 클릭합니다.

## 6. 설교 원고(칼럼) 관리

설교 원고를 리치 텍스트 에디터(Tiptap)로 작성하고 관리하는 메뉴입니다. 작성된 원고는 공개 페이지의 칼럼 섹션에 매거진 스타일로 표시됩니다.

### 공개 페이지 기능

- **검색** -- 제목, 성경구절, 본문 내용으로 칼럼을 검색할 수 있습니다.
- **정렬** -- 최신순, 오래된순, 조회순, 제목순으로 정렬할 수 있습니다.
- **조회수** -- 각 칼럼의 조회수가 자동으로 추적되어 카드 하단에 표시됩니다.
- **매거진 레이아웃** -- 칼럼 상세 페이지는 히어로 이미지, 작성자 메타 정보, 풀 쿼트, 관련 칼럼 등을 포함한 매거진 스타일로 표시됩니다.

### 칼럼 목록 확인

1. 사이드바에서 **설교 원고**를 클릭합니다.
2. 작성된 칼럼 목록이 표시됩니다.

### 새 칼럼 작성

1. 설교 원고 페이지에서 **+ 새 원고** 버튼을 클릭하거나, 대시보드의 **+ 설교 원고 작성** 바로가기를 사용합니다.
2. 아래 항목을 입력합니다:

| 항목 | 필수 여부 | 설명 |
|------|-----------|------|
| 제목 | 필수 | 칼럼 제목을 입력합니다. |
| 성경 구절 | 선택 | 관련 성경 구절을 입력합니다. |
| 연결 설교 | 선택 | 드롭다운에서 기존 설교를 선택하여 연결합니다. |
| 커버 이미지 URL | 선택 | 칼럼 대표 이미지 URL을 입력합니다. |
| 본문 | 필수 | 리치 텍스트 에디터를 사용하여 본문을 작성합니다. |

3. **발행하기** 버튼을 클릭합니다.

### 리치 텍스트 에디터(Tiptap) 사용법

에디터 상단의 도구 모음에서 다양한 서식을 적용할 수 있습니다:

| 버튼 | 기능 | 설명 |
|------|------|------|
| **B** | 굵게 | 선택한 텍스트를 굵게 표시합니다. |
| *I* | 기울임 | 선택한 텍스트를 기울임꼴로 표시합니다. |
| H2 | 제목 2 | 선택한 텍스트를 큰 제목으로 변환합니다. |
| H3 | 제목 3 | 선택한 텍스트를 중간 제목으로 변환합니다. |
| 점 기호 | 글머리 기호 목록 | 글머리 기호가 있는 목록을 만듭니다. |
| 1. | 번호 매기기 목록 | 번호가 매겨진 목록을 만듭니다. |
| 따옴표 | 인용문 | 선택한 텍스트를 인용문 블록으로 표시합니다. |

### 서식 적용 방법

1. 서식을 적용할 텍스트를 마우스로 드래그하여 선택합니다.
2. 원하는 도구 버튼을 클릭합니다.
3. 버튼이 활성화 상태(진한 색상)로 변하면 해당 서식이 적용된 것입니다.
4. 같은 버튼을 다시 클릭하면 서식이 해제됩니다.

### 칼럼 수정

1. 칼럼 목록에서 수정하려는 항목의 **수정** 링크를 클릭합니다.
2. 내용을 수정합니다.
3. **수정하기** 버튼을 클릭합니다.

### 칼럼 삭제

1. 칼럼 목록에서 삭제하려는 항목의 **삭제** 버튼을 클릭합니다.
2. 확인 대화상자에서 **확인**을 클릭합니다.

## 7. 갤러리 관리 (라이트박스)

교회 사진을 업로드하고 관리하는 메뉴입니다. 업로드된 사진은 공개 페이지의 갤러리에 표시됩니다.

### 갤러리 확인

1. 사이드바에서 **갤러리**를 클릭합니다.
2. 업로드된 사진이 격자 형태로 표시됩니다.

### 사진 업로드

1. 갤러리 페이지 상단의 **사진 업로드** 영역을 사용합니다.
2. 아래 항목을 입력합니다:

| 항목 | 필수 여부 | 설명 |
|------|-----------|------|
| 제목 | 선택 | 사진 제목을 입력합니다. (입력하지 않으면 "제목 없음"으로 표시) |
| 카테고리 | 선택 | 사진 카테고리를 입력합니다. (예: 예배, 행사, 교회) |
| 이미지 | 필수 | 업로드할 이미지 파일을 선택합니다. |

3. **업로드** 버튼을 클릭합니다.
4. 업로드가 완료되면 사진이 갤러리에 즉시 추가됩니다.

### 지원 이미지 형식

- JPG/JPEG
- PNG
- GIF
- WebP
- 기타 일반 이미지 형식

### 사진 삭제

1. 삭제하려는 사진 위에 마우스를 올립니다.
2. 사진 오른쪽 상단에 빨간색 **X** 버튼이 나타납니다.
3. 해당 버튼을 클릭합니다.
4. 확인 대화상자에서 **확인**을 클릭합니다.

### 라이트박스 사용 (공개 페이지)

공개 교회 소개 페이지(`/about`)에서 사진을 클릭하면 전체화면으로 볼 수 있습니다.
- **← →** 화살표 버튼 또는 키보드 방향키로 이전/다음 사진 이동
- **ESC** 키 또는 닫기 버튼으로 닫기

## 8. 교회 일정 관리

월간 일정을 등록하고 공개 페이지에 캘린더 형태로 표시하는 메뉴입니다.

### 일정 목록 확인

1. 사이드바에서 **교회 일정**을 클릭합니다.
2. 등록된 일정이 날짜 내림차순으로 표시됩니다.

### 새 일정 등록

1. 오른쪽 상단의 **+ 일정 등록** 버튼을 클릭합니다.
2. 아래 항목을 입력합니다:

| 항목 | 필수 여부 | 설명 |
|------|-----------|------|
| 제목 | 필수 | 일정 제목 (예: 부활절 연합예배) |
| 구분 | 선택 | 일반 / 예배 / 특별 / 모임 |
| 시작일 | 필수 | 일정 시작 날짜 |
| 종료일 | 선택 | 여러 날에 걸친 일정의 종료 날짜 |
| 설명 | 선택 | 일정 상세 내용 |

3. **저장** 버튼을 클릭합니다.

### 일정 삭제

- 목록에서 해당 일정의 **삭제** 버튼을 클릭합니다.

### 공개 페이지 (`/calendar`)

- 월간 CSS Grid 캘린더로 표시됩니다.
- 이전/다음 달 이동 가능.
- 일정을 클릭하면 상세 팝업이 표시됩니다.
- 페이지 하단에 이달 전체 일정 목록도 표시됩니다.

## 9. 기도 요청 관리

성도들이 제출한 기도 제목을 확인하고 응답 처리하는 메뉴입니다.

### 기도 요청 목록 확인

1. 사이드바에서 **기도 요청**을 클릭합니다.
2. 접수된 모든 기도 제목이 최신순으로 표시됩니다.
   - 익명 여부, 제출 날짜, 내용 확인 가능

### 응답 처리

1. 응답된 기도 제목 오른쪽의 **응답 처리** 버튼을 클릭합니다.
2. 해당 기도 제목이 "응답됨" 상태로 변경됩니다.
3. 익명이 아닌 경우 공개 페이지(`/prayer`)의 "응답된 기도" 섹션에 표시됩니다.
4. 취소하려면 **응답 취소** 버튼을 클릭합니다.

### 공개 페이지 (`/prayer`)

- 기도 제목 제출 폼 (이름/익명 선택, 내용 입력)
- 응답된 기도 목록 (비익명만 공개)

## 10. 사이트 설정

### 설정 페이지 접속

### 변경 가능한 항목

- 유튜브 채널 URL과 블로그 URL은 홈페이지의 빠른 링크 섹션에 표시됩니다.
- 예배 시간 정보는 홈페이지의 예배 안내 섹션에 표시됩니다.
- 교회 주소는 홈페이지의 카카오맵 위치 표시에 사용됩니다.

## 11. 자동 동기화 (YouTube / 네이버 블로그 / Google Drive)

교회 유튜브 채널의 설교 영상, 네이버 블로그의 설교 칼럼, Google Drive의 찬양 콘티가 매일 자동으로 동기화됩니다.

### 자동 동기화 스케줄

- **실행 시간**: 매일 오전 6시 (Vercel Cron)
- **YouTube 동기화**: 채널의 최신 영상을 자동으로 설교 목록에 추가
- **블로그 동기화**: 네이버 블로그의 최신 글을 자동으로 설교 칼럼에 추가
- **Google Drive 동기화**: 지정된 폴더의 찬양 콘티 파일을 자동으로 다운로드/업로드

### 수동 동기화

대시보드에서 아래 버튼을 클릭하면 즉시 동기화를 실행할 수 있습니다:

- **유튜브 동기화** -- YouTube 채널에서 최신 설교 영상 가져오기
- **블로그 동기화** -- 네이버 블로그에서 최신 설교 칼럼 가져오기
- **Google Drive 동기화** -- Google Drive에서 최신 찬양 콘티 가져오기

### 참고 사항

- 이미 등록된 영상/글/파일은 중복으로 추가되지 않습니다.
- 동기화된 칼럼의 조회수는 0에서 시작하여 방문자가 열람할 때마다 증가합니다.
- Google Drive의 `.key` 파일은 로컬 배치 스크립트(`scripts/sync-drive.mjs`)로 PDF 변환 가능합니다.
- 캐시(ISR)로 인해 동기화 후 최대 5분 내에 공개 페이지에 반영됩니다.

## 자주 묻는 질문

### 로그인이 안 됩니다.

- 이메일과 비밀번호를 정확히 입력했는지 확인하세요.
- 기본 계정은 `admin@dongnam.church` / `admin1234` 입니다.
- 문제가 지속되면 관리자에게 문의하세요.

### 파일 업로드가 실패합니다.

- 파일 크기가 50MB를 초과하지 않는지 확인하세요.
- 지원되는 파일 형식인지 확인하세요 (PDF, HWP, DOCX, PPTX, PPT).
- 인터넷 연결 상태를 확인하세요.

### 삭제한 콘텐츠를 복구할 수 있나요?

- 삭제된 콘텐츠(설교, 칼럼, 콘티, 주보, 사진)는 **복구할 수 없습니다**.
- 삭제 전에 반드시 확인 대화상자의 내용을 읽어주세요.

### 홈페이지에 변경 사항이 바로 반영되지 않습니다.

- 브라우저의 캐시를 삭제하거나, 새로고침(Ctrl+Shift+R 또는 Cmd+Shift+R)을 시도하세요.
- 설정 변경은 저장 후 즉시 반영되지만, 브라우저 캐시로 인해 지연될 수 있습니다.
